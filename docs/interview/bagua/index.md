# AI Agent 面试八股文

这一页不是七类能力速查的重复版本，而是另一条更适合系统复习的“八股文”路径。它把高频问题按专题重新排成一条连续阅读线，方便你先补回答骨架，再回到分类页做针对性查漏。

和 [面试题专区总览](/interview/) 的区别是：

- 七类路径更像速查导航，适合先定位自己卡在哪一类能力
- 八股文路径更像系统阅读，适合把相关问题连起来背、连起来讲

## 适合谁读

- 你已经知道大概有哪些面试题，但回答还不够成体系
- 你想把分散概念整理成一套更稳定的口头表达顺序
- 你准备集中复习几轮，不想每次都从分类页重新跳转

## 怎么用这条路径

不要试图第一次就从头读完所有模块。更实用的方式是先挑一两个最薄弱的专题，整理短答版，再回来补追问方向和模块之间的关系。读这条路径时，优先关注“定义、边界、取舍、风险”四件事，而不是机械背结论。

## 阅读建议

### 如果你是初学者

建议先按这条顺序推进：基础概念 -> 大模型基础 -> 工具调用 -> 核心框架 -> 记忆系统 -> RAG 技术 -> Prompt 工程 -> 多智能体 -> 工程化实践。前半段先把系统边界和执行主链路说顺，后半段再补复杂场景和生产判断。

### 如果你已经做过项目

可以优先从自己最容易被追问的专题切入。一个更实用的顺序通常是：工程化实践 -> RAG 技术 -> 多智能体 -> 记忆系统，然后再回补基础概念、核心框架和 Prompt 工程，把表达收束得更完整。
